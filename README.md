# ai_phishing_detection-defense_system
Final year Project: A web based application that is impowerd with Ai and machine learning model for real time phinshing detection of scam URLs and Emails. Further integration of chrome extension.


# Features 
Detect and block phishing URLs and Emails

# Tech Stack
python , react.js , Flask 

# Installation steps
 open terminal and create a virtual environment of python 
 for frontend:
 npm run dev
 for backend:
 python app.py
